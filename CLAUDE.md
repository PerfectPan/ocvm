# Claude Instructions

This repository uses `AGENTS.md` as the canonical agent instruction file.

Before making changes:

1. Read `AGENTS.md`.
2. Follow the contribution and validation workflow in `CONTRIBUTING.md`.
3. Keep changes scoped to the task.
4. Do not commit local config, credentials, generated logs, temporary workspaces, build artifacts, or machine-specific paths.

For substantial changes, write or update an RFC under `rfcs/`.
